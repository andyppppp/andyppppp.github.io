Andy Pearson

Question 1

Define what a multimedia system is and make reference to both multimedia and computer systems to formulate your answer.

Criteria Marks
Describes what multimdeia means 1/2
Describes what a computer system is 1/2
Defines what a multimedia system is with reference to both previous aspects 1

Sample answer:
Multimedia can be defined as any application that combines text with graphics, animation, audio, video, and/or virtual reality. A computer system is a combination of hardware, software and people organized to perform a function. Therefore, a multimedia system is simply a system capable of displaying different forms of data with the help of hardware, software and people.
